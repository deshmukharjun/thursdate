## 🚀 Tech Stack

### Frontend
- **React.js** (Vite)
- **Tailwind CSS** for styling
- **React Router** for navigation
- **Axios** for HTTP requests

### Backend
- **Node.js**
- **Express.js**
- **MongoDB** (via Mongoose ODM)
